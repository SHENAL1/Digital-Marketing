Requirements

There are system requirements in order to install and setup Imroz theme and its components properly. Make sure that you are running the latest version of WordPress, PHP version 5.6 or higher and MySQL version 5.6 or higher. We also recommend the following PHP configuration limits. https://wordpress.org/about/requirements/

Recommended PHP configuration limits

                                        
                                            max_execution_time = 300;
                                            upload_max_filesize = 128M;
                                            post_max_size = 128M;
                                            max_input_time = 300;
                                            memory_limit = 256M;

Installing the theme

Important! if you download the All Files archive from ThemeForest and try to upload it to your WordPress, you will receive this error: Theme is missing the style.css stylesheet. Please make sure you unzipped the main archive and are uploading the correct file to your WordPress.

When you are ready to install a theme, you must first upload the theme files and then activate the theme itself. The theme files can be uploaded in two ways:

How to Setup and Demo Import (Video): https://youtu.be/ClCmn8ueYp0

1. Using Theme Uploader

Follow the steps as instructed in the images below:

  1. From the dashboard go to AppearanceThemesAdd New Upload Theme
  2. Click on “Choose File” and select imroz.zip
  3. Click on “Install Now”.
  4. Active the Theme

2. Using FTP Clients

Follow the following steps to upload your theme using FTP clients:

  1. Use an FTP such as FileZilla and go to your WordPress installation director
  2. Navigate to /wp-content/themes/ folder and upload the theme folder there
  3. Log in to your WP Admin Dashboard and open the AppearanceThemes menu
  4. You will now see Imroz Theme listed among the other themes. Click on Activate and you’re done!

3. Updating the theme

We recommend that you use the Envato Market plugin to keep the theme up-to-date.

  1. Download the Envato Market plugin from here: http://envato.github.io/wp-envato-market/dist/envato-market.zip
  2. Install the plugin from PluginsAdd New
  3. Once installed, you should see an Envato Market menu on the admin sidebar. Click on it.
  4. Enter your Envato API Personal Token in the Token field. If you don’t have an API token you can create one here or click on the generate a personal token link on that page.
  5. Once you have pasted your token in the Token field, click on Save Changes.
  6. Should your token be correct, two extra tabs are now available on your Envato Market page. Click on the Plugins tab.

If an update is available for the Imroz Theme, click on the Update Available link, which should start installing your update.

Envato Market Plugin Official Site: https://envato.com/market-plugin/

4. Installing required plugins

Imroz theme requires a set of core and additional plugins to function correctly. A plugin is an additional component that offers functionality and features beyond a typical WordPress installation. Without them, most of its functionality is not available. After installing the theme you will be prompted to install the required plugins at the top of your dashboard.

  1. From the dashboard go to Appearance Install Plugins
  2. Toggle all the required plugins and hint “Install”;
  3. The last step will be activating the plugins.

Importing Demo Contents

  • One Click Import
  • Manually Import

One Click Method

Want to build your site like the demo? You can do it by only one click! Imroz an easy solution for the demo content you saw on our preview website. Please follow the instructions below:

Note: Please remember that when you are done with the demo import, your some existing settings will be replaced by the demo contents Like: Your menu, Customizer. So it is recommended that you should only do this in fresh site.

  1. Ensure, that the One-Click Import plugin is installed and activated;
  2. From the dashboard go to Appearance Import Demo Data
  3. Choose the demo you want to install and click on the “Import” button​ below​;
  4. And change the permalink: From the dashboard go to Settings Permalink

Manually Import

Please follow the instructions below:

  1. Click switch to manual import from right corner.
  2. Choose content.xml file for all content import
  3. Choose widget.wie file for widget import
  4. Choose customizer.dat file for customizer import
  5. Choose redux_options.dat file for theme options import. Redux option name: “imroz_options”
  6. and last click Import demo data button

Note: We have added all the above files to the Theme package -> Demo Data folder.

One page Navigation

  1. Prepare the page
  2. Create a new page. To do that go to Pages Add New and give the page a title and then add a Block element to the content. Add section unique "ID" into Per Block.
  3. Prepare the menu
  4. Go to Appearance Menu click the create a new menu link Give the new menu a unique name and then click the Create Menu button.
  5. Add menu item with section ID
  6. Select Onepage menu for this page from page attributes options

Creating a portfolio page

To create a page with a portfolio template and add it to your website:

  1. From the dashboard go to Pages Add New
  2. Enter your portfolio page title
  3. If you want to overwrite global options. Like Header, Footer etc style. Please customize here.
  4. Click the Publish Button
  5. Select "Elementor Full Wdth" Page Template
  6. Add a section with "full width" and "No Gap"
  7. Choose and drop a Portfolio Projects element from the left side
